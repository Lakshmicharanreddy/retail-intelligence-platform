# Retail Intelligence Platform

## Overview

An end-to-end retail analytics platform that simulates real-time retail data, processes it through an ETL pipeline, stores it in PostgreSQL, predicts future demand using Machine Learning, and visualizes KPIs using Tableau.

## Tech Stack

- Python
- PostgreSQL
- Pandas
- NumPy
- SQLAlchemy
- Scikit-Learn
- Tableau
